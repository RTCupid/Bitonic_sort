### Running the program
Repository cloning, build and compilation is performed using the following commands:

```
git clone git@github.com:RTCupid/Bitonic_sort.git
cd Bitonic_sort
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
```
